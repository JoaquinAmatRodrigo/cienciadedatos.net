Introducción¶


Una serie temporal (time series) es una sucesiÃ³n de datos ordenados cronolÃ³gicamente, espaciados a intervalos iguales o desiguales.

El proceso de Forecasting consiste en predecir el valor futuro de una serie temporal, bien modelando la serie temporal Ãºnicamente en funciÃ³n de su comportamiento pasado (autorregresivo) o empleando otras variables externas a la serie temporal.

A lo largo de este documento, se describe cÃ³mo utilizar modelos de regresiÃ³n de Scikit-learn para realizar forecasting sobre series temporales. En concreto, se hace uso de Skforecast, una librerÃ­a sencilla (en construcciÃ³n) que contiene las clases y funciones necesarias para adaptar cualquier modelo de regresÃ³n de Scikit-learn a problemas de forecasting.

Forecasting¶


Cuando se trabaja con series temporales, raramente se quiere predecir solo el siguiente elemento de la serie ($t_{+1}$), sino todo un intervalo futuro o un punto alejado en el tiempo ($t_{+n}$).

Dado que, para predecir el momento $t_{+2}$ se necesita el valor de $t_{+1}$, y $t_{+1}$ se desconoce, es necesario hacer predicciones recursivas en las que, cada nueva predicciÃ³n, se basa en la predicciÃ³n anterior. A este proceso se le conoce como recursive forecasting o multi-step forecasting y es la principal diferencia respecto a los problemas de regresiÃ³n convencionales.


Forecasting autorregresivo con scikit learn¶

La principal adaptaciÃ³n que se necesita hacer para aplicar modelos de scikit learn a problemas de forecasting es transformar la serie temporal en un matriz en la que, cada valor, estÃ¡ asociado a la ventana temporal (lags) que le preceden.

TranformaciÃ³n de una serie temporal en una matriz de 5 lags y un vector con el valor de la serie que sigue a cada fila de la matriz.

Importancia predictores¶


Dado que el objeto ForecasterAutoreg utiliza modelos scikit-learn, una vez entrenado, se puede acceder a la importancia de los predictores. En regresores GradientBoostingRegressor() o RandomForestRegressor(), la importancia de los predictores estÃ¡ basada en la reducciÃ³n de impureza y es accesible mediante el mÃ©todo get_feature_importances().
